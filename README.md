sino-xenic-phonology-analysis
Quantitative analysis of Sino-Japanese and Sino-Vietnamese phonological adaptation using the Unihan database. Utilizing Python and pandas

Overview

This project examines phonological correspondence patterns between Sino-Japanese (On-yomi) and Sino-Vietnamese (Hán Việt) readings using data from the Unihan database.

The focus is not on 1-to-1 sound mapping, but on how two distinct linguistic systems adapted Middle Chinese source forms in different ways.

Both On-yomi and Hán Việt reflect historical Middle Chinese layers rather than modern Mandarin. Tone information from pinyin is not used in this analysis.

Core Observations

Japanese and Vietnamese both inherit readings from Middle Chinese, but through different adaptation processes.

Japanese phonology compressed several historical finals (e.g., -ng → -n).

Many Middle Chinese finals are reflected in Japanese as long vowels (e.g., こう, しょう).

Vietnamese retains tonal distinctions and may show greater apparent variation at the character level.

Some apparent diversity in Vietnamese may reflect tonal or orthographic variation rather than true phonemic diversity.

Project Goal

Rather than repeating established linguistic descriptions, this project aims to identify quantitative "hotspots" of divergence:

Where does Japanese show higher reading diversity?

Where does Vietnamese show unexpected variation?

Are differences systematic at the character level?

Can phonological compression or expansion be measured statistically?

The goal is to combine linguistic theory with structured data analysis to reveal measurable patterns.
